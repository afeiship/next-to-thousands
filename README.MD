#to-thousands
> To  thousands for number.


## usage:
+ install:
```bash
npm install afeiship/to-thousands --save

# OR
yarn install afeiship/to-thousands
```

+ show code:
```javascript
import toThousands from 'to-thousands';

toThousands('10000')  //10,000
toThousands('100')    //100
toThousands(10000)    //10,000
toThousands(1000,'|') //1|000
```
