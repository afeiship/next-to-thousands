# next-to-thousands
> To  thousands for number.

## installation
```bash
npm install -S @feizheng/next-to-thousands
```

## usage
```js
import '@feizheng/next-to-thousands';

nx.toThousands('10000')  //10,000
nx.toThousands('100')    //100
nx.toThousands(10000)    //10,000
nx.toThousands(1000,'|') //1|000
```
