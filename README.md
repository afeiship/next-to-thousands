# next-to-thousands
> To thousands for number.

## install:
```bash
npm install -S afeiship/next-to-thousands --registry=https://registry.npm.taobao.org
```

## usage:
```javascript
import toThousands from 'to-thousands';

toThousands('10000')  //10,000
toThousands('100')    //100
toThousands(10000)    //10,000
toThousands(1000,'|') //1|000
```
